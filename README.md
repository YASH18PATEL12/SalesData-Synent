# Superstore Sales Dataset - Sales Data Analysis

## Objective
The objective of this project was to analyze sales performance and generate business insights using the Superstore Sales dataset.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook / VS Code

## Dataset Information
The dataset contained the following key fields:

- Ship Mode
- Segment
- Country
- City
- State
- Postal Code
- Region
- Category
- Sub-Category
- Sales
- Quantity
- Discount
- Profit

## Tasks Performed

### 1. Data Loading and Exploration
- Imported the dataset using Pandas.
- Examined dataset structure using:
  - `head()`
  - `info()`
  - `describe()`

### 2. Sales Analysis
- Calculated total sales by product category.
- Visualized category-wise sales performance using bar charts.
- Identified categories generating the highest revenue.

### 3. Top-Selling Products Analysis
- Grouped data by Sub-Category.
- Calculated total sales for each sub-category.
- Identified the top-selling sub-categories.
- Visualized the top 10 selling sub-categories using bar charts.

### 4. Profit Analysis
- Calculated overall business profit.
- Analyzed profit contribution by category.
- Identified the most profitable sub-categories.
- Visualized profit distribution using charts.

### 5. Regional Analysis
- Compared profit across different regions.
- Identified high-performing and low-performing regions.

### 6. Segment Analysis
- Analyzed sales distribution among customer segments.
- Visualized segment contribution using pie charts.

## Business Insights
- Certain categories generated significantly higher revenue than others.
- Top-selling sub-categories contributed a major portion of total sales.
- Profitability varied across categories and regions.
- Customer segments showed different levels of sales contribution.

## Outcome
The project successfully generated business insights related to sales performance, product profitability, customer segments, and regional performance, supporting data-driven decision making.
